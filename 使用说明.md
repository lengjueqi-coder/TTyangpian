# 人像 Prompt 生成器 PRO - 使用说明

## 首次启动

1. 双击 `启动.command` 文件
2. 首次运行会自动创建虚拟环境并安装依赖（约30秒）
3. 安装完成后自动打开浏览器访问 http://localhost:5800
4. 如果 macOS 提示"无法验证开发者"，请右键点击 → 打开 → 确认

## 日常使用

- 双击 `启动.command` 即可启动
- 关闭终端窗口将停止服务
- 浏览器访问地址：http://localhost:5800

## 数据导入导出

- 点击顶部"导出"按钮，下载 zip 数据包
- 发给朋友后，他点击"导入"选择 zip 文件即可同步所有数据

## 系统要求

- macOS
- Python 3.9+（通常系统自带）
